## Supported sources and formats
- YouTube
- SoundCloud
- Spotify
- HTTP URLs

## Formats
- MP3
- FLAC
- WAV
- Matroska/WebM (AAC, Opus or Vorbis codecs)
- MP4/M4A (AAC codec)
- OGG streams (Opus, Vorbis and FLAC codecs)
- AAC streams
- Stream playlists (M3U and PLS)

# Commands

A full list of commands for use with NinoMusic
